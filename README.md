# URL Shortener

A full-stack URL shortening service built with:
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB

## Features
- Shorten long URLs
- Custom URL support
- User authentication
- Click tracking

## Setup
1. Clone the repository
2. Install dependencies:
```bash
# Backend
cd backend
npm install

# Frontend
cd frontend
npm install
```

3. Create .env files in both backend and frontend directories
4. Run the application:
```bash
# Backend
npm run dev

# Frontend
npm run dev
```
